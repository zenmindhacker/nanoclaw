## PHASE 1 — WINTER / MENSTRUAL
**Days 1–6 · Your dates this cycle: May 27 – Jun 1**

| Discipline | Phase Expression |
|---|---|
| **O.W. Phase** | Reflective |
| **Medical Phase** | Menstrual — uterine lining releases |
| **Hormones** | Progesterone & estrogen at lowest |
| **Season** | Winter |
| **Moon Phase** | Dark Moon |
| **Light** | Dark |
| **Element** | Earth |
| **TCM Element** | Water |
| **TCM / WP Via** | Yin Dominant (inward, feminine) · Via Negativa |
| **Archetype** | Crone / Wise Woman |
| **Goddesses** | Hecate (Greek), Nephthys (Egyptian), Kali (Hindu), Baba Yaga (Slavic), Cerridwen (Celtic), Persephone in descent, Black Madonna |
| **Keywords** | Darkness, wisdom, gestation, stillness, vision |
| **Emotion** | Introspective, dreamy, sensitive, intuitive, spiritually connected |
| **Energy** | Reflective, slow, containing, internalized, spiritual |
| **Libido** | Usually little desire for physical intimacy — more spiritual connection |
| **Physical** | Cramping, back ache, migraine, faintness, exhaustion, tearfulness. Lowest pain threshold. Ligaments stiffen as estrogen lowers — prone to injury. Vivid dreams. |
| **Seed Cycling** | Flaxseeds & pumpkin seeds |
| **Tea** | Chamomile, red raspberry leaf, dandelion root |
| **Food** | Iron-rich: dark leafy greens, nuts, beef, chicken, clams, dark chocolate. Anti-inflammatory: omega-3s, antioxidants. Warm, cooked foods. Avoid cold/raw. |
| **Outward Action** | Retreat. Only do what is essential. Delay decisions & stressful appointments. Dream, rest, slow down. |
| **Relationships** | Desires solitude or quiet communion with women. Does not want to be around demands. |
| **Affirmation** | *I sink into my depths and listen to my dreams.* |
| **Vaginal Discharge** | Blood |
| **Weird Science** | Vivid dreams day 1–2. Lowest pain threshold. Most prone to injury (ligaments). |
| --- | --- |
| **🆕 Ayurveda** | **Vata dominant** (air + ether — movement, release, lightness). Qualities: cold, light, mobile, irregular. Support with: sesame oil massage (abhyanga), warm nourishing soups, ginger & turmeric, routine, early bedtime. Avoid: cold food, raw salads, stimulants, irregular schedule. |
| **🆕 Chakra** | **Root (Muladhara)** — safety, grounding, survival, release. Colour: red. Practice: walking barefoot, hip-opening yoga, grounding meditation, red foods. |
| **🆕 Tarot** | **Suit of Pentacles** (Earth, body, resources, slow growth). Card archetype: **The High Priestess** — inner knowing, mystery, the unseen, trusting what cannot yet be spoken. |
| **🆕 Nervous System** | **Dorsal vagal** tendency — body moves toward deep rest and conservation. Not collapse, but protective stillness. This is biological, not weakness. Honour it. Avoid forcing yourself into high-stimulation environments. |
| **🆕 Exercise** | Rest days 1–2. Gentle walking, restorative yoga, yin yoga from day 3–4. Avoid HIIT or heavy lifting. Body is doing enormous work already. |
| **🆕 Cognitive Style** | Right brain dominant. Intuitive, pattern-recognizing, big-picture. Vivid and symbolic dreams carry insight. Not ideal for detailed analytical work — ideal for visioning, journalling, letting answers surface. |
| **🆕 Skin** | Dull, potentially dry and sensitive. Lowest estrogen = less collagen support and hydration. Focus: gentle cleanser, rich hydrating serum, face oils (rosehip, marula), avoid harsh exfoliation or actives. |

## PHASE 2 — SPRING / FOLLICULAR
**Days 7–13 · Your dates this cycle: Jun 2–8**

| Discipline | Phase Expression |
|---|---|
| **O.W. Phase** | Dynamic |
| **Medical Phase** | Follicular — egg follicle ripening, estrogen rising, uterine lining building |
| **Hormones** | Oestrogen rising |
| **Season** | Spring |
| **Moon Phase** | Waxing |
| **Light** | Lightening |
| **Element** | Air |
| **TCM Element** | Wood |
| **TCM / WP Via** | Yang Dominant (outward energy, masculine) · Via Positiva |
| **Archetype** | Virgin / Maiden |
| **Goddesses** | Brigid (Celtic), Ostara (Germanic), Flora (Roman), Idun (Norse — keeper of youth), Saraswati (Hindu — creativity/learning), Persephone returning, Aphrodite emerging |
| **Keywords** | New beginnings, dynamic, exuberance, self-confident |
| **Emotion** | Calm, open, dynamic, clear, energetic, enthusiastic, able to cope with irritations |
| **Energy** | Rising dynamic — growing outward |
| **Libido** | Rising — carefree |
| **Physical** | Egg follicle ripening. Most empathy (lower progesterone = brain's control over bad habits weakest). High ability to read others' emotions. |
| **Seed Cycling** | Flaxseeds & pumpkin seeds |
| **Tea** | Dandelion root tea |
| **Food** | Extra water. Complex carbs: fruit, whole grains, starchy vegetables. Light, fresh foods. |
| **Outward Action** | Start projects. Clear visioning. Fresh start. Organize and prioritize. Spring cleaning — clearing out what slipped during menstruation. |
| **Relationships** | Easy going, trusting, outgoing |
| **Affirmation** | *I step forward in action with a lightness of heart.* |
| **Vaginal Discharge** | Sticky / none |
| **Weird Science** | Highest empathy and emotion-reading ability. Brain's habit control is weakest — watch addictive patterns. |
| --- | --- |
| **🆕 Ayurveda** | **Kapha building** (earth + water — stability, growth, nourishment), transitioning toward Pitta as ovulation approaches. Support with: lighter foods, movement, stimulating exercise, bitter greens, lemon water. Avoid: heavy/oily foods, oversleeping. |
| **🆕 Chakra** | **Sacral (Svadhisthana)** — creativity, pleasure, flow, emotional intelligence. Colour: orange. Rising toward **Solar Plexus (Manipura)** — confidence, identity, personal power. Practice: hip circles, dance, creative projects, journalling desires. |
| **🆕 Tarot** | **Suit of Wands** (Fire, action, inspiration, new ventures). Card archetype: **The Fool** (pure potential, new beginning) or **The Star** (hope, renewal). |
| **🆕 Nervous System** | **Ventral vagal rising** — social engagement system activating. Open, curious, connected, regulated. Window of opportunity to have brave conversations, try new things, build connections. |
| **🆕 Exercise** | Best phase for HIIT, strength training, trying new workouts. Higher pain threshold, faster recovery. Estrogen supports muscle building and connective tissue recovery. Ideal: lift heavy, sprint, try something new. |
| **🆕 Cognitive Style** | Left brain rising — analytical, planning, detail-oriented, strategic. Excellent for learning new skills, making decisions, starting focused work, research. |
| **🆕 Skin** | Clearest, most resilient skin of the cycle. Estrogen boosts collagen and hyaluronic acid. Best time for: exfoliation (AHAs/BHAs), active treatments, facials, experimenting with new products. Natural glow building. |

## PHASE 3 — SUMMER / OVULATORY
**Days 14 · Your dates this cycle: ~Jun 9**

| Discipline | Phase Expression |
|---|---|
| **O.W. Phase** | Expressive |
| **Medical Phase** | Ovulatory — egg released from ovary into fallopian tube |
| **Hormones** | LH surge triggers ovulation. Estrogen peaks, progesterone begins rising. |
| **Season** | Summer |
| **Moon Phase** | Full Moon |
| **Light** | Full bright light |
| **Element** | Water |
| **TCM Element** | Fire |
| **TCM / WP Via** | Yang Dominant (outward energy, masculine) · Via Positiva |
| **Archetype** | Mother |
| **Goddesses** | Gaia, Isis (Egyptian), Aphrodite (Greek), Shakti/Devi, Lakshmi, Yemaya (Yoruba — mother of waters/fertility), Oshun (Yoruba — love, sensuality, abundance), Mother Mary |
| **Keywords** | Fertility, radiating, caring |
| **Emotion** | Loving, nurturing, nourishing, sustaining, energized, connected |
| **Energy** | Full, sustaining — losing sense of self in work or mothering |
| **Libido** | Full, peak libidinous desire |
| **Physical** | Egg released. Voice higher pitched. Attracted to artistic men. Shopping more. High propensity for disgust (biological protection). Possible ovulatory cramping or mid-cycle spotting. |
| **Seed Cycling** | Flaxseeds & pumpkin seeds |
| **Tea** | Green tea, cinnamon |
| **Food** | Vitamin D foods: dairy, eggs, salmon. Omega-3s: flaxseed, chia, walnuts. Antioxidants: blueberries, blackberries, dark leafy greens. |
| **Outward Action** | Work hard, love well. Birth creative projects. Stay up late. Receptive to others' input. Harmony and connection. |
| **Relationships** | Loving, giving, nurturing. Reach out to friends, family, partner. |
| **Affirmation** | *I embrace my life with love and generate beauty around me.* |
| **Vaginal Discharge** | Clear and stretchy, like egg white. Very wet feeling. |
| **Weird Science** | Voice highest pitched. Biologically magnetized to attract. Shopping impulses high. |
| --- | --- |
| **🆕 Ayurveda** | **Pitta dominant** (fire + water — transformation, intensity, heat). High energy, clarity, drive. Support with: cooling foods (coconut, cucumber, mint, coriander), hydration, avoid overheating. Reduce: caffeine, alcohol, spicy food, overworking. |
| **🆕 Chakra** | **Heart (Anahata)** — love, compassion, giving, connection. Colour: green. **Throat (Vishuddha)** — authentic expression, communication, speaking truth. Practice: heart-opening yoga, singing, speaking from the heart, breathwork. |
| **🆕 Tarot** | **Suit of Cups** (Water, emotions, love, relationships, intuition). Card archetype: **The Empress** — abundance, fertility, sensuality, creative power, nurturing the world. |
| **🆕 Nervous System** | **Full ventral vagal** — optimal social engagement. Maximum empathy, presence, connection, co-regulation. Best window for leadership, visibility, important conversations. |
| **🆕 Exercise** | Peak performance window. Highest coordination and stamina. Good for: competitions, personal bests, group workouts, challenging hikes. Body can sustain more intensity and recovers faster. |
| **🆕 Cognitive Style** | **Whole-brain integration** — analytical and intuitive simultaneously. Best for: presentations, negotiations, collaborative strategy, public speaking, pitching ideas. Whole-picture thinking at its highest. |
| **🆕 Skin** | Peak glow and radiance. Maximum estrogen = highest collagen, plumpness, luminosity. Some people experience mid-cycle breakout (estrogen surge triggers brief sebum increase). If so: light salicylic spot treatment. Otherwise: enjoy your skin, keep routine simple. |

## PHASE 4 — AUTUMN / LUTEAL
**Days 15–27 · Your dates this cycle: Jun 10–22**

| Discipline | Phase Expression |
|---|---|
| **O.W. Phase** | Creative |
| **Medical Phase** | Luteal — corpus luteum forms, progesterone rises, uterine wall prepares |
| **Hormones** | Progesterone rising then falling. Estrogen falling. |
| **Season** | Autumn |
| **Moon Phase** | Waning |
| **Light** | Darkening |
| **Element** | Fire |
| **TCM Element** | Metal |
| **TCM / WP Via** | Yin Dominant (inward, feminine) · Via Negativa |
| **Archetype** | Enchantress / Wild Woman |
| **Goddesses** | Hecate (approaching), Kali (Hindu), Lilith (Hebrew — reclaimed power), Medusa (sovereignty), Cerridwen (Celtic), Banbha (Irish), Demeter (harvest/loss), The Morrigan (Irish — truth, battle, prophecy) |
| **Keywords** | Magical, witchy, destructive, intuitive |
| **Emotion** | Creative, emotional, sensitive. Shadow material surfaces. Truth-teller. |
| **Energy** | Waning dynamic — descending inward |
| **Libido** | Peaks and troughs — can be very intense |
| **Physical** | Lowered immune system. Voice may be crackly. Most psychic time. Toward end: cramping, back ache, bloating, tiredness, tender breasts, sugar/carb cravings, mood swings. Vivid dreams. |
| **Seed Cycling** | Sesame & sunflower seeds |
| **Tea** | Peppermint, ginger |
| **Food** | Healthy fats: fish, nuts, avocado. Extra protein-rich snacks. Reduce sugar. Magnesium-rich foods for PMS: dark chocolate, leafy greens, pumpkin seeds. |
| **Outward Action** | Wind up project action. Begin to reflect and assess. Focus on inner-directed creative projects. Deal with issues and problems. |
| **Relationships** | Needs to balance dynamic interactions with focused, energized creative time alone |
| **Affirmation** | *I use the sword of my intolerance to cut deep and true. I keep hold of my vision and manifest it.* |
| **Vaginal Discharge** | None / blobby thick and yellow |
| **Weird Science** | Most psychic phase. Emotional sensitivity peaks as hormones decline. Low tolerance for inauthenticity — BS detector is sharpest here. |
| --- | --- |
| **🆕 Ayurveda** | **Early luteal: Pitta** (progesterone = heat, intensity). **Late luteal: Vata rising** (things beginning to disassemble, irregularity, anxiety). Support with: routine, grounding, warm cooked foods, reducing stimulants, sesame oil massage, early bedtime, minimal screen time before sleep. |
| **🆕 Chakra** | **Third Eye (Ajna)** — intuition, inner vision, discernment, seeing through illusion. Colour: indigo. Late luteal moves toward **Crown (Sahasrara)** — dissolution of ego, connection to mystery, heightened sensitivity to everything. Practice: meditation, dream journalling, oracle cards, intuitive writing. |
| **🆕 Tarot** | **Suit of Swords** (Air, mind, truth, conflict, clarity, cutting through illusion). Card archetype: **The Hermit** — inner retreat, wisdom-seeking, the lantern turned inward. |
| **🆕 Nervous System** | **Sympathetic activation increases** toward end of phase. Triggers feel closer to the surface. Most important phase for nervous system regulation tools: breathwork, cold water, somatic movement, journalling, boundaries. What feels intolerable late luteal is often accurate — the filter is down, not broken. |
| **🆕 Exercise** | Moderate intensity: pilates, barre, yoga, swimming, walking. Body temperature slightly elevated (progesterone effect) — avoid overheating. Strength gains still possible early luteal. Lower intensity as period approaches. |
| **🆕 Cognitive Style** | Right brain, creative, synthesizing. Shadow material surfaces — excellent for: deep creative work, journalling, therapy, pattern recognition across your life. Reduced tolerance = honesty. Clarity of what isn't working is a gift, not dysfunction. |
| **🆕 Skin** | Progesterone increases sebum → potential clogged pores and hormonal breakouts (chin/jaw zone). Late luteal: possible puffiness and dehydration. Early luteal: clay masks, salicylic acid, keep pores clear. Late luteal: prioritize hydration, reduce harsh actives, gentle anti-inflammatory skincare. |
